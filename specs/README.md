This directory defines the forward-looking product specs for Forward Roll using plain markdown.

- [[architecture]] documents the plugin boundary, runtime constraints, and packaging model.
- [[workflow]] documents bootstrap flow, planning loops, review boundaries, and jj-oriented workflow intent.

## Authoring

Treat these files as the aspirational product description for the branch. They should describe the plugin we want to ship, the workflow boundaries we intend to preserve, and the runtime constraints that follow from shipping only static skills and scripts.
